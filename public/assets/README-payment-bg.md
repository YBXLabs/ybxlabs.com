# Payment Background Image

Please add your `payment-bg.png` file to this directory (`public/assets/`).

## Requirements:
- File name: `payment-bg.png`
- Recommended size: 1920x1080px or higher
- Format: PNG, JPG, or WebP
- The image will be used as a full-screen background for the payment page

## Usage:
The image is referenced in the payment page template at `/pay/[amount]` and will be displayed as a full-screen background with an overlay for the payment form.
